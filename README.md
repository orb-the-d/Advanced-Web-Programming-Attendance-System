# Advanced Web Programming - Student Attendance System

A complete web-based student attendance tracking system built for the Advanced Web Programming course (Tutorial 1 & Tutorial 2).

##  Project Overview

This project implements a fully functional student attendance management system with features including:
- Student registration form with validation
- Interactive attendance tracking with clickable checkboxes
- Real-time absence and participation calculation
- Color-coded student status (Green/Yellow/Red)
- Search and sorting functionality
- Statistical reporting
- Responsive design



### Tutorial 1 - HTML5, CSS3
-  **Exercise 1**: Student attendance table with structured layout
-  **Exercise 2**: Add student form with validation fields
-  **Exercise 3**: Professional navigation bar

### Tutorial 2 - JavaScript & jQuery
-  **Exercise 1**: Automatic absence/participation counting with color-coded rows
-  **Exercise 2**: Complete form validation (Student ID, names, email)
-  **Exercise 3**: Dynamic student addition without page reload
-  **Exercise 4**: Statistical report generation
-  **Exercise 5**: Interactive hover effects and click-to-view info
-  **Exercise 6**: Highlight excellent students animation
-  **Exercise 7**: Search by name & sorting by absences/participation

##  Technologies Used

- **HTML5** - Semantic structure and markup
- **CSS3** - Modern styling with CSS variables and animations
- **JavaScript (ES6)** - Core interactivity and logic
- **jQuery 3.6.0** - DOM manipulation and event handling
